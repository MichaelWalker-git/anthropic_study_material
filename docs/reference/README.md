# Reference documentation

Offline snapshots of the official Claude / MCP documentation, used as source
material for the CCA-F mock-exam question bank. Each file is the Markdown export of
the page at the URL listed below, fetched on 2026-06-17.

These are reference copies — they are not part of any build and may drift from the live
docs over time. Re-fetch by appending `.md` to the page URL (the docs sites serve a
Markdown variant at that path).
